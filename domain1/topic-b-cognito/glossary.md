- **User Pool**: user directory that authenticates users and issues tokens.
- **App client**: configuration that defines how an app uses the user pool (OAuth settings).
- **Hosted UI**: Cognito-managed login page that runs OAuth flows for you.
- **OAuth Authorization Code**: flow that returns an authorization code, then exchanges it for tokens.
- **Scopes**: permissions/claims requested during login (openid, email, profile).
